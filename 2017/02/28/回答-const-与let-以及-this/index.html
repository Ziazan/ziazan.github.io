1.关于this

this 的指向是由他所在函数调用的上下文决定的，而不是由它所在的函数定义的上下文决定的。
（在函数内部，this的值取决于函数是如何调用的。）

2.关于const 与 let的选择

  • const 与 let 都是用来声明变量的
  • let: 声明变量。 在可以使用var 声明的变量都可以考虑使用let 避免变量被重复定义。
  • const :声明常量。如果这个值在后面都不会改变，建议使用const 声明成常量。如： const PI = 3.1415926 这种。
