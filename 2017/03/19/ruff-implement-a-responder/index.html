按键抢答器

使用三个按键模块和两个板载LED灯。

  • 如果按键 A 按下，板载蓝灯（led-b）亮；
  • 按键 B 按下，板载红灯（led-r）亮；
  • 按键 C 用于重置抢答器，C 键按下，两灯都灭；
  • 按键 A 或 B 按下后，直到按下按键 C 重置抢答器，即使另一按键按下，两LED 灯也不会有变化。
